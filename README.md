# Yong Mei Pricing WebApp

Ứng dụng tính giá tem nhãn dạng static web app.

## Cách up lên GitHub Pages

1. Tạo repository mới trên GitHub.
2. Upload toàn bộ file trong thư mục này lên repository.
3. Vào **Settings > Pages**.
4. Ở phần **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save và chờ GitHub Pages publish.
6. Web app sẽ chạy bằng file `index.html`.

## Chạy local

Mở trực tiếp file `index.html` bằng trình duyệt.
